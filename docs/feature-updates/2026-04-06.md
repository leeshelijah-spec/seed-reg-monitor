## 실행 환경 복구

- `scripts/resolve_python.ps1`를 추가해 실행 가능한 base Python을 찾아 `.venv`를 복구하거나 재사용하도록 정리
- `.venv`가 손상되었을 때 base Python으로 가상환경을 다시 만들고 `requirements.txt`를 재설치하는 흐름을 보강
- `scripts/start_dashboard.ps1`를 단일 시작 진입점으로 정리하고 기본 실행, 헬스 체크, 테스트 기준으로 동작을 검증

## 뉴스 피드백 및 그룹 검토 흐름 개선

- 기사 피드백을 영향도, 긴급도, 관련 여부, 잡음 여부 기반의 구조화된 입력으로 개편
- 같은 기사 재수집 시 이전 피드백을 다시 활용하도록 피드백 학습 경로와 수집 로직을 추가
- 관련 기사 묶음 보기, 확대 토글, 묶음 저장과 단독 저장을 같은 화면 흐름에서 처리하도록 대시보드 UI를 확장

## 뉴스 그룹 규칙 보강

- 제목 토큰 비교만 사용하던 묶음 규칙에 제목 3-gram 유사도와 제목+요약 문맥 토큰 비교를 추가
- 표현이 조금 다른 동일 이슈 기사도 같은 그룹으로 묶일 수 있도록 규칙 임계치를 보완
- 잡음 저장 시 영향도와 긴급도 값이 비전송되어 400이 발생하던 문제를 수정하고 회귀 테스트를 추가
